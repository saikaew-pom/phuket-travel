This file tracks foundational mandates, architectural decisions, and the current state of the project.

## 🎯 Core Mandates
- **Vanilla First**: All interactivity must be implemented in Vanilla JS. No heavy frameworks (React/Vue/Angular) in the final build.
- **Performance**: Maintain 95+ Lighthouse score. Minimize external scripts.
- **SEO Integrity**: Every page must have valid JSON-LD and unique meta tags.
- **Responsive Design**: Glass-morphism UI must be fully functional on mobile and desktop.
